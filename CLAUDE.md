## 项目简介
琉玥纪是一个水晶手串品牌。要做一个**微信小程序**，让用户在线用3D水晶珠子DIY设计手串并购买。

## 竞品参考
- "养个石头"（MineStone）微信小程序
- 竞品截图在 `/养个石头界面图/` 目录下
- 关键差异：我们要做**真3D渲染**（Three.js），养个石头是2D贴图模拟

## 核心功能
1. **3D手串设计器**（核心卖点）— 选珠 + 排列 + 实时预览 + 总价计算
2. **成品商城** — 品牌现成的手串直接购买
3. **设计广场** — 用户分享设计方案，互相参考
4. **订单流程** — 设计→支付→制作→实物图确认→发货

## 技术方向（供开发参考）
- 前端：Next.js + React + Tailwind CSS
- 3D：Three.js + React Three Fiber
- 后端：Supabase / Node.js
- 小程序方案：H5 + web-view 嵌入微信小程序
- 支付：微信支付
