## 一、项目概述

### 1.1 项目背景
琉玥纪是一个水晶手串品牌，同时也是水晶供应商。当前销售主要依赖线下和传统电商渠道，客户无法直观体验水晶搭配效果。需要建设一款小程序，让客户能在线上自主选择水晶珠子、设计专属手串并完成购买，提升用户体验和转化率。

### 1.2 项目目标
- 为客户提供 **3D可视化手串定制** 体验，降低决策门槛
- 打通 **选珠→设计→下单→发货** 全链路闭环
- 通过设计社区（设计广场）促进用户分享，带动自然增长
- 建立品牌的数字化私域流量入口

### 1.3 目标用户
| 用户类型 | 描述 | 核心诉求 |
|---------|------|---------|
| C端消费者 | 喜欢水晶、手串的年轻女性为主（18-35岁） | 选好看的珠子搭配出独一无二的手串 |
| 送礼用户 | 为朋友/伴侣定制手串作为礼物 | 有设计感、有仪式感 |
| 水晶爱好者 | 对水晶功效、品质有一定了解 | 按功效/材质精准选择 |

### 1.4 竞品参考
- **养个石头（MineStone）**：微信小程序，2D贴图模拟3D效果，DIY设计器+成品商城+设计广场
- 竞品详细分析见附件：`/竞品分析/养个石头分析.md`

## 三、核心功能详细设计

### 3.1 3D手串设计器（核心页面）

#### 3.1.1 页面布局

```
┌─────────────────────────────┐
│  ← 返回    琉玥纪    使用须知  │
│  [手围: 14cm]  [总价: ¥206]  │
├─────────────────────────────┤
│                             │
│     ╭─── 3D手串预览区 ───╮   │
│     │                   │   │
│     │   (可旋转/缩放)    │   │
│     │    珠子排列在      │   │
│     │    3D圆环上       │   │
│     │                   │   │
│     ╰───────────────────╯   │
│                             │
│  [🗑删除] [💾保存]  [🛒完成设计] │
├─────────────────────────────┤
│  🔍 搜索水晶名称...          │
│                             │
│  白水晶 │ ◯净体白水晶 6mm ¥3 │
│  紫水晶 │ ◯净体白水晶 8mm ¥5 │
│  黄水晶 │ ◯净体白水晶10mm ¥10│
│  粉水晶 │ ◯奶白晶   8mm ¥4 │
│  ...   │ ...              │
└─────────────────────────────┘
```

#### 3.1.2 交互规则

| 交互 | 行为 | 说明 |
|------|------|------|
| 点击珠子卡片 | 添加一颗珠子到手串 | 珠子出现在圆环下一个空位 |
| 点击手串上的珠子 | 选中该珠子（高亮） | 可删除或替换 |
| 手势旋转 | 3D手串整体旋转 | 可从不同角度查看效果 |
| 双指缩放 | 放大/缩小手串 | 查看细节 |
| 长按珠子 | 查看该珠子的实物图 | 弹窗展示实拍照片 |
| 删除按钮 | 删除选中的珠子 | 总价实时更新 |
| 保存 | 保存当前设计到"我的设计" | 可暂存多个方案 |
| 完成设计 | 跳转到定制商品详情页 | 确认手围→查看明细→下单 |

#### 3.1.3 手围逻辑

| 手围(cm) | 状态 | 提示 |
|---------|------|------|
| < 13 | 过小 | "手围过小，请继续添加珠子" |
| 13-19 | 正常 | 显示当前手围值 |
| > 19 | 过大 | "手围偏大，建议减少珠子或更换小尺寸" |

手围计算公式：`手围 = Σ(每颗珠子直径) × π / π = Σ(每颗珠子直径)`
> 实际为所有珠子直径之和，因为珠子紧密排列在弹力绳上

#### 3.1.4 价格计算
- 总价 = Σ(每颗珠子单价)
- 运费规则：（待定，需确认包邮策略）
- 无额外手工费/串珠费

### 3.2 定制商品详情页（完成设计后）

#### 3.2.1 页面内容
1. **手串3D渲染图**（与设计器中一致）
2. **总价**
3. **手围选择/确认**（支持修改，带"检查手围"按钮）
4. **定制明细表**

| 字段 | 示例 |
|------|------|
| 材料 | 乌拉圭紫水晶 10mm |
| 单价 | ¥24 |
| 数量 | 1 |
| 金额 | ¥24 |

5. **收货地址**
6. **备注**（给商家留言，限45字）
7. **购前须知**
   - 手围确认提醒
   - 制作时长（约2个工作日）
   - 实拍确认（制作完成后发实物图，确认后发货）
   - 退换规则（48小时鉴赏期）
   - 运费说明

#### 3.2.2 底部操作栏
- 客服（跳转在线客服）
- 加购物车
- 立即购买

### 3.3 商城（成品商品）

#### 3.3.1 分类体系
- 按水晶种类分类（紫水晶、粉水晶、兔毛水晶...）
- 相关服务（鉴定证书等）
- 水晶周边（收纳、清洁用品等）

#### 3.3.2 商品详情页
- 商品图片轮播
- 商品名称 + 价格
- 商品详情（规格、所属分类、文案描述）
- 费用明细 + 收货地址 + 备注
- 底部：客服 / 加购物车 / 立即购买

### 3.4 设计广场

#### 3.4.1 频道
| 频道 | 内容来源 |
|------|---------|
| 设计师款 | 品牌官方设计的搭配方案 |
| 优秀客订 | 精选客户的定制方案（需客户授权） |
| 热门排行 | 按使用人数排序 |

#### 3.4.2 设计卡片信息
- 手串渲染图
- 设计名称
- 设计者昵称
- 使用人数
- 点击进入 → 查看实物图 + 设计细节 + "使用此设计"一键复制到设计器

## 四、数据模型（供开发参考）

### 4.1 核心实体

```
珠子(Bead)
├── id
├── name          // 名称（如"巴西黄水晶"）
├── category      // 分类（黄水晶）
├── size_mm       // 尺寸（6/8/10/12mm）
├── price         // 单价
├── thumbnail     // 缩略图
├── texture_3d    // 3D材质贴图
├── real_photo    // 实物照片
└── description   // 描述/功效

设计方案(Design)
├── id
├── user_id
├── name          // 设计名称
├── beads[]       // 珠子列表（有序）
│   ├── bead_id
│   └── position  // 在手串上的位置
├── total_price
├── circumference // 手围(cm)
├── is_public     // 是否公开到设计广场
├── use_count     // 被使用次数
└── created_at

订单(Order)
├── id
├── user_id
├── type          // "custom"定制 / "standard"成品
├── design_id     // 关联设计方案（定制订单）
├── items[]       // 订单项
├── total_amount
├── shipping_fee
├── address_id
├── status        // 待付款/已付款/制作中/待确认/已发货/已完成
├── remark
└── created_at
```

## 五、业务流程

### 5.1 定制下单流程

```
选珠设计 → 完成设计 → 确认手围 → 确认明细 → 填写地址
    → 支付 → 商家制作 → 发送实物图 → 用户确认
    → 发货 → 收货 → 完成（48h鉴赏期）
```

### 5.2 订单状态流转

```
待付款 ──支付──→ 已付款/待制作 ──制作完成──→ 待确认（实物图）
  │                                          │
  │                                    ┌─────┴─────┐
  │                                 确认发货    拒绝(重做)
  │                                    │
  │                                 已发货 ──收货──→ 已完成
  │
  └──超时未支付──→ 已取消
```
